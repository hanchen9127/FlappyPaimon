# Flappy Paimon

**Author:** Hanchen Wang  

## Overview
*Flappy Paimon* is a 2D arcade game where the player navigates Paimon through randomly generated barriers, using vertical and horizontal movement, a flash skill, and avoiding collisions.

## Features
- Vertical & horizontal movement, flash skill with shadow indicator.  
- Random barriers with variable gaps.  
- Scoreboard and timer.  
- Background music and sound effects.  
- Quick restart with **R** key.

## Controls
| Key          | Action                       |
|--------------|------------------------------|
| Space / Up   | Fly up                       |
| Left / Right | Move horizontally            |
| X            | Flash forward                |
| R            | Quick restart                |

## How to run
```bash
git clone <repo-url>
pip install pygame
python main.py
```
